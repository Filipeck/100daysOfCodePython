# 100 Days of Code: The Complete Python Pro Bootcamp

## Sobre o repositório

Neste repositório irei documentar todo o meu progresso no curso [100 Days of Code: The Complete Python Pro Bootcamp](https://www.udemy.com/course/100-days-of-code/).

### Objetivo dos Desafios

Os desafios propostos serão uma ótima maneira de reforçar todo o aprendizado e progressão que tenho adquirido com Python, indo desde o básico até a aplicação do Python de forma mais profissional.
As pastas serão organizadas em numeração progressiva e com o nome do projeto que será desenvolvido, tendo como objetivo apresentar as seguintes temátcas:

- **Fundamentos do Python**: Conceitos básicos de Python, como variáveis, tipos de dados e operadores.
- **Estruturas de Dados**: Trabalhar com listas, tuplas, dicionários e conjuntos de forma eficiente.
- **Funções e Módulos**: Criação de funções reutilizáveis, organizado o código de maneira mais estruturada.
- **Programação Orientada a Objetos**: Iremos abordar o funcionamento de classes e objetos.
- **Manipulação de Arquivos**: Iremos ler e escrever arquivos.
- **Bibliotecas e Frameworks**: Exploraremos bibliotecas populares como NumPy, Pandas, Matplotlib e frameworks como Flask e Django. Além de Selenium, Beautiful Soup, Request, Scikit Learn e Plotly.
- **Projetos do Mundo Real**: Construção de projetos práticos como um Tinder auto swiper, jogo da cobrinha, site de blog, envio automático de candidatura no LinkedIn, entre outros.

### Conclusão

Estou muito animado para começar esta jornada e documentar meu progresso ao longo desses desafios. Tenho certeza de que este curso será transformador e me ajudará a alcançar meus objetivos como desenvolvedor Python. Vamos lá!
